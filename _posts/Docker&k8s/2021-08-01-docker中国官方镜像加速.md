## docker中国官方镜像加速

> https://www.docker-cn.com/registry-mirror

安装好Docker后增加以下内容，然后重启Docker服务为了永久性保留更改，您可以修改下边的文件并添加上 registry-mirrors 键值.，本地镜像也可以修改该值

```bash
# vim /etc/docker/daemon.json

{
  "registry-mirrors": ["https://registry.docker-cn.com"]
}

```
