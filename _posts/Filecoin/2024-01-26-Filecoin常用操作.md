本文档介绍 Filecoin常用操作 的相关内容。

## 查询所以扇区过期时间

```text
lotus-miner sectors check-expire

```

## 查询旷工号的版本

```ini
lotus-miner state sector [矿工号] [sectorsid]
lotus-miner state sector f0719301 0

```

## 续期从多少天开始到多少天结束的扇区至统一高度

```bash
使⽤ from + to 筛选过期时间满⾜我们设定条件的扇区，对它们进⾏调整到 540 天的周期，它会去⾃动设定这些扇区的分别的到期时间:

# --from 默认为当前时间+120(1 hour) <from> defaults to: now + 120 (1 hour) (default: 0)

# --to 默认为当前时间+32天 <to> defaults to: now + 92160 (32 days) (default: 0)

# --extension默认为540天1555200

lotus-miner sectors renew --from 2219472 --to 2256392 --extension 1555200

```
