本文档介绍 ubuntu 升级 的相关内容。

## 更新系统软件包

```sql
apt-get update -y
apt-get upgrade -y
apt-get dist-upgrade -y

```

## 删除不用的软件包和清理缓存

```bash
apt-get autoremove
apt-get clean

```

## 重启后开始升级

```text
do-release-upgrade

```
