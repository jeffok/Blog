本文档介绍 清理iptables规则 的相关内容。

## 彻底清理iptables规则(openstack会存在dhcp异常)

```bash
yum install -y iptables-services
/usr/libexec/iptables/iptables.init stop

```

http://xstarcd.github.io/wiki/Linux/iptables_forward_internetshare.html

```bash
转发进来局域网：
iptables -t nat -A PREROUTING -d 1.2.3.4/32 -i eth0 -p tcp -m tcp --dport 3389 -j DNAT --to-destination 192.168.247.148:3389

当作网关上外网：
iptables -t nat -A POSTROUTING -o eth0 -j SNAT --to-source 1.2.3.4
或者指定来源ip段：
iptables -t nat -A POSTROUTING -s 192.168.108.0/24 -o eth0 -j SNAT  --to-source 1.2.3.4

```
