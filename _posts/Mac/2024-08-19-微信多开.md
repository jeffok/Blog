本文档介绍 微信多开 的相关内容。

```bash
# 源码地址：<a href="https://github.com/Sunnyyoung/WeChatTweak-macOS">https://github.com/Sunnyyoung/WeChatTweak-macOS</a>

# 安装方法：<a href="https://github.com/Sunnyyoung/WeChatTweak-macOS/wiki">https://github.com/Sunnyyoung/WeChatTweak-macOS/wiki</a>

<a href="https://juejin.im/post/5c0494026fb9a049dd80085b">https://juejin.im/post/5c0494026fb9a049dd80085b</a>

准备工作
xcode 安装
下载代码
git clone https://github.com/Sunnyyoung/WeChatTweak-macOS.git
编译
cd WeChatTweak-macOS
sudo make install
安装完成后，重启微信可启用插件

问题：
如果微信升级，出现
xcrun: error: invalid active developer path (/Library/Developer/CommandLineTools), missing xcrun at: /Library/Developer/CommandLineTools/usr/bin/xcrun.
执行以下命令
xcode-select --install
然后再运行安装sudo make install
如果还未解决，再执行
sudo xcode-select -switch /

点击右键登陆新账号

```
