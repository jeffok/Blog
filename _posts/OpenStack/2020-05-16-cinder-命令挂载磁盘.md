本文档介绍 cinder 命令挂载磁盘 的相关内容。

```sql
创建云硬盘
cinder create --display-name 'user for db' --volume-type CEPH-SATA 20
创建与虚拟机在同一主机的云硬盘
cinder create --hint local_to_instance=24234352-394a-4edd-89f5-315e5a5b8a4a --volume-type sas --name bjdz-qa-example.compile01_disk2 500

参数 	描述
Display-name 	用于定义该云盘的显示名称
Volume-type 	用于定义使用哪个 cinder 后端进行云盘创建
20 	代表创建一个 20GB 的云盘

```

```bash
查看云磁盘ID
cinder list --display-name 'user for db'
根据名称查看

挂载云盘
nova volume-attach 7dcfee9a-9338-489b-a335-215b8441d67b d1e1194f-d33b-4169-9ce2-6644fa111d89
利用 nova volume-attach 云主机id 云盘id 进行挂载

卸载云硬盘
nova volume-detach 7dcfee9a-9338-489b-a335-215b8441d67b d1e1194f-d33b-4169-9ce2-6644fa111d89

X 	云主机 	云盘
ID 	7dcfee9a-9338-489b-a335-215b8441d67b 	d1e1194f-d33b-4169-9ce2-6644fa111d89

```
