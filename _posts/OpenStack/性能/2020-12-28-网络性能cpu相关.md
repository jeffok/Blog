> https://blog.csdn.net/wxyq_q/article/details/88651339  设置cpu为性能模式

1. 修改cpu为性能模式

```bash
# 设置开机启动

echo '/usr/bin/cpupower frequency-set -g performance' >> /etc/rc.local

# 手动执行立即生效

/usr/bin/cpupower frequency-set -g performance

```

2. 修改OpenStack物理机内核参数

```bash
# 设置内核参数

sed -i 's/^GRUB_CMDLINE_LINUX=.*/GRUB_CMDLINE_LINUX="rhgb quiet kvm-intel.nested=1 intel_iommu=on"/g' /etc/default/grub

# 更新内核参数

grub2-mkconfig -o /boot/grub2/grub.cfg

```
