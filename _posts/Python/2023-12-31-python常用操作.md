## Python 生成pyo/pyc

```bash
# 生成pyc

python -m py_compile 12.py

# 生成pyo

python -O -m py_compile 12.py

```

## 判断ip是否在子网中

```python
import IPy

 >>>'192.168.1.100' in IPy.IP('192.168.1.0/24')
 is   True
 >>>'192.168.1.0/24' in IPy.IP('192.168.0.0/16')
 is   True
 >>>'192.168.1.0/24' in IPy.IP('192.168.0.0-192.168.8.255')
 is   True

```

## python中运行shell命令

```python
# Method 1

import subprocess
string = subprocess.check_output("COMMAND_STRING", [shell=True]).decode()

# Method 2

import os
result = os.system("COMMAND_STRING")

```
