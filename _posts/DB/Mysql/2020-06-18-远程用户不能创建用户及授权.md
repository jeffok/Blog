本文档介绍 远程用户不能创建用户及授权 的相关内容。

## 创建数据库用户

```sql
grant all privileges on *.* to 'openstack'@'%' identified by 'uWZrszSay9puwFX' with grant option;

```

## 修改权限

```sql
GRANT PROXY ON ''@'%' TO 'openstack'@'%' WITH GRANT OPTION;
```

> **注意**：如果不增加该条记录，不能远程创建用户并授权。

## 刷新权限

```sql
FLUSH PRIVILEGES;
```

## 查看用户权限

```sql
show grants for root@"%";

```

## 更新用户权限

```sql
update user set grant_priv='Y' where user='root';
select user,host,grant_priv from user;

```

##### 查看用户权限情况

```sql
select * from user where User='openstack' \G;

```
